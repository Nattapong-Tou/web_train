การใช้ filter แบบ sample

{% comment %} Filter default ของ x คือ {{ x|default:1 }}
default if None คือ {{ none|default_if_none:2 }}
Filter Add String คือ {{ str|add:str }}
Filter Add Integer is {{ int|add:100 }}
{% if str|length_is:0 %} String is empty.. {% else %} Length string is {{ str|length }} {% endif %} {% endcomment %}

{% endif %} {% if number2|divisibleby:2 %} ตัวแปร {{number2}} หารด้วย 2 ลงตัว
{% else %} ตัวแปร {{number2}} หารด้วย 2 ไม่ลงตัว
{% endif %} การใช้ filesizeformat {{ file_size }} byte >> {{file_size|filesizeformat}}
{{int}} >> {{int|floatformat}}
{{int}} >> {{int|floatformat:2}}
{{int}} >> {{int|floatformat:1}}
{%endcomment%}
